h1. smd_section_roles

h2. Usage

# Install the plugin.
# Visit Admin>Prefs:Section roles.
# Choose sections that each user level can post in.
# Save the prefs.

Results:

* When anyone visits the Write panel they can only publish articles to one of the nominated sections.
* Users can only see articles on the Articles list panel in sections they're permitted to post.
* Users can only assign articles via multi-edit to sections they have access.
* Users can only navigate next/prev between articles in sections to which they've been given access.

h2. Known issues

* Breaks the _Override form_ selector.
* Doesn't (yet) restrict the multi-edit section assignment after submission.
